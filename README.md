# Matrix Operations Tool

## Description

This project is a Matrix Operations Tool developed using Python and the NumPy library. It allows users to perform basic matrix operations through an interactive command-line interface.

## Features

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose
- Determinant Calculation

## Technologies Used

- Python
- NumPy

## How to Run

1. Install NumPy

```
pip install numpy
```

2. Run the project

```
python matrix_operations.py
```
